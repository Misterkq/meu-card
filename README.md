Este projeto foi criado com base na inspiração visual de um card de perfil de uma pessoa que vi na internet. Ao pesquisar mais sobre, percebi que a página original havia sido gerada por uma ferramenta de Inteligência Artificial. Pensei: por que não tentar recriar esse mesmo conceito do zero, escrevendo o código manualmente?

A partir dessa ideia, desenvolvi do zero a estrutura, o design e as interações do card, utilizando aprendizado prático de desenvolvimento web e manipulação de DOM.

---

## Recursos e Funcionalidades

- Efeito 3D Tilt: O card acompanha a movimentação do ponteiro do mouse em tempo real com inclinação calculada via JavaScript.
- Sistema de Copiar Nickname: Ao clicar nos itens de redes sociais, o texto é copiado para a área de transferência com uma notificação visual ("Copiado!").
- Animação de Neve (Snowflakes): Efeito contínuo de partículas caindo pelo fundo da tela gerado dinamicamente com JavaScript e animações CSS.
- Player de Áudio Integrado: Música de fundo com controle manual de play/pause no canto da tela e ativação no primeiro clique da página para respeitar os navegadores.
- Layout Responsivo e Personalizado: Estilização em CSS com fundo temático, transparência glassmorphism e alinhamento centralizado.
----
Na parte de JavaScript utilizei um pouco de IA por falta de conhecimento, e algumas parte de css tambem
----

## Tecnologias Utilizadas

- HTML5: Estruturação dos componentes da página.
- CSS3: Animações, posicionamento flex/relative/absolute, keyframes e efeitos de iluminação e ilusão 3D.
- JavaScript (Vanilla): Manipulação de eventos do DOM, cálculos de posição do mouse, controle da API de área de transferência e reprodução de áudio.
